About This Project

The purpose of this project was to analyze weather patterns as they related to the equator. The data was harvested from the OpenWeatherMap API utilizing over 500+ randomly selected cities in order to formulate the snapshot you see here.

Once the data was pulled and cleaned, MatPlotLib was utilized to plot the data in various juxtapositions. Factors that were examined regarding the equator included the following: temperature, cloudiness, wind speed, and humidity. This website provides access to the visualizations of the project and source data used to formulate those same visualizations.

Visualizations - Latitude vs. Cloudiness

The plot above shows us what we would expect when analyzing climates near the equator, a mirrored effect. There are a lot of similarities with cities on exact opposite ends of the equator.

Visualizations - Latitude vs. Temperature

This is where the data lines up with preconceived notions--the average temperature rises drastically as we approach the equator and the further away we go, we see winter-like conditions.

Visualizations - Latitude vs. Wind Speed

Once again we see data that is spread out without any real strong correlation. There seems to be bunching from a latitude of 20-50, but much of that may be related to the randomized selection of cities, as opposed to any real evidence of something bigger.
